Key Features

Our system offers a comprehensive solution for attendance management

QR Code Generation

Generate unique QR codes for each class session with customizable expiration times.

Geolocation Verification

Ensure students are physically present in class with location-based verification.

Attendance Analytics

Generate detailed reports and visualize attendance patterns with interactive charts.

How It Works

Simple and efficient attendance tracking in three easy steps

1

Lecturer Generates QR Code

Lecturers create a unique QR code for each class session with a specified expiration time.

2

Students Scan QR Code

Students scan the QR code using their mobile devices while physically present in class.

3

Attendance is Recorded

The system verifies location and automatically records attendance in the database.

About Afe Babalola University

Afe Babalola University, Ado-Ekiti (ABUAD) is a private university located in Ado-Ekiti, Ekiti State, Nigeria. The university was founded by Aare Afe Babalola, a renowned lawyer and philanthropist.

This QR-code attendance management system is designed specifically for ABUAD to streamline the attendance tracking process and ensure accurate records for all courses.
